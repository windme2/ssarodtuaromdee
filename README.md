# 🚐 Rodtuaromdee - Premium Van Service

A dedicated platform for **professional van rental services with experienced drivers**.
We provide safe, comfortable, and reliable transportation across Thailand, reflecting our commitment to excellent service and customer satisfaction.

---

## ⚙️ Tech Stack

- **Frontend:** Next.js (App Router, SSG), React, TypeScript
- **Styling & UI:** TailwindCSS, Lucide React, Framer Motion
- **Internationalization:** Custom i18n with middleware routing (TH/EN/ZH)
- **Analytics:** Vercel Analytics
- **Image Optimization:** Sharp (build-time), Next/Image (runtime), WebP format

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ and npm

### Clone the repository

```bash
git clone https://github.com/windme2/rodtuaromdee.git
cd rodtuaromdee
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
# Frontend will be available at http://localhost:3000
```

### Build for production

```bash
npm run build
npm run start
```
